# AI Banking Customer Churn Prediction

## Project Overview
This project builds an Artificial Intelligence system to predict customer churn in the banking sector using machine learning techniques. 

Customer churn prediction is a critical task for banks to identify customers who are likely to leave the bank and take preventive actions through targeted marketing strategies.

The model analyzes multiple customer attributes such as credit score, geography, age, account balance, and product usage to predict the probability of churn.

## Objectives
- Predict whether a customer will leave the bank
- Identify key factors influencing churn
- Help banking institutions improve customer retention
- Demonstrate the use of AI in financial analytics
